# Personal Portfolio Website

## Overview

This project is a personal developer portfolio website created to showcase my technical skills, development projects, and background as a Computer Science student.

The portfolio serves as a digital resume where visitors can explore my work, learn about the technologies I use, and connect with me professionally.

The website focuses on a clean design, responsive layout, and clear presentation of projects and skills.

## Installation

Clone the repository:

git clone https://github.com/prernaparashari/portfolio-website.git

Navigate to the project folder:

cd portfolio-website

Install dependencies:

npm install

Start the development server:

npm start
